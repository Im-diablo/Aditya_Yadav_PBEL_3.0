How It Works

ML Module (50%): XGBoost analyses 30 ULB transaction features. V1–V28 are PCA-anonymised. Trained with scale_pos_weight=577 to handle 577:1 class imbalance without oversampling.

NLP Module (30%): DistilBERT scans the description for phishing patterns, suspicious URLs, and urgency language.

Fusion: Weighted average of active modules. Missing inputs auto-rebalance weights to sum to 1.0.

Siamese ResNet18

Trained on the CEDAR Signature Dataset — 55 signers × 24 genuine + 24 forged = 2,640 images.

The Siamese architecture learns to compare two signatures rather than classify each independently, ideal for small datasets where pairs far outnumber individual samples.

How distance works:
Small distance → similar embeddings → GENUINE
Large distance → dissimilar embeddings → FORGED

Architecture Details

Shared-weight ResNet18 backbone pretrained on ImageNet.
Layers 1 & 2 frozen to preserve low-level feature extraction.
256-dim embedding head with Dropout 0.3.
Contrastive Loss with margin = 1.0.
